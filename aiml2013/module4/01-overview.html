What's in This Module

  1. Readings & Prep: A conceptual introduction to the attention mechanism (3Blue1Brown), feature extraction with pre-trained CNNs (PyImageSearch), similarity metrics for image retrieval, and the ChromaDB quickstart. Together, these explain why deep features outperform hand-crafted descriptors, how to evaluate retrieval quality, and how to set up the vector database for the lab.
  2. Module 4 Lab: Image Similarity Search with ChromaDB. Build a complete image retrieval system: extract MobileNetV2 embeddings, store in ChromaDB, query by image, compute precision@k, analyze failure cases, and compare against HOG on the same images.
  3. Combined Lab: The Universal Search Engine. For dual-enrolled students taking both NLP and CV. One notebook with a ChromaDB instance that indexes both image embeddings and text chunks, queryable in both modalities.
  4. Assignment Sheet: Deliverable details, abbreviated rubrics with Canvas links, and the due date for Tuesday, April 21.
  5. Ethics Discussion: This week's ethics prompt on visual search and surveillance.
