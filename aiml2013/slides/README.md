# aiml2013/slides/ — AIML 2013 Lecture Slides

Slide decks and visual assets for in-class presentations, organized by week.

## Structure

```
slides/
└── week1/
    ├── week1-slides.pptx    ← Week 1 lecture slides
    ├── 1bit.html            ← Interactive: 1-bit color demo
    ├── 4bit.html            ← Interactive: 4-bit color demo
    ├── 24bit.html           ← Interactive: 24-bit color demo
    └── rgb-cartoon.png      ← RGB channel diagram
```

The HTML files in `week1/` are interactive browser visualizations used during lecture to demonstrate color depth and pixel representation. Open them directly in a browser.

Add new weeks as `week N/` subdirectories following the same pattern.

## Notes

Slides are not part of the build pipeline — they are not processed or uploaded by `build.sh`. Share slides with students directly via Canvas Files or a linked Google Slides URL.
