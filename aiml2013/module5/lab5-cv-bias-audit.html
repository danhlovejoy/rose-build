The Task

In the readings and in class, you learned that a model's overall accuracy can mask large disparities across subgroups. A face-attribute classifier that is 94% accurate overall might be 99% accurate on lighter-skinned males and 65% accurate on darker-skinned females. The overall number looks good; the per-group numbers tell a different story.

This lab uses a pre-trained image classifier and a dataset with demographic labels to run a structured bias audit. You will compute standard classification metrics (accuracy, precision, recall, F1) on the full dataset, then disaggregate those metrics by demographic group to find where the model fails disproportionately. The lab is less about code and more about what the numbers reveal — the same classifier, the same test set, but a completely different conclusion depending on how you slice the results.

Dataset: CelebA Gender Classification

You will use a subset of the CelebA (Celebrity Faces Attributes) dataset for a binary gender classification task. CelebA provides 40 attribute labels per image, including "Male" (binary). You will use a pre-trained classifier to predict this attribute and then audit its accuracy across subgroups defined by other attributes (e.g., "Young" vs. "not Young", "Wearing_Hat" vs. "not Wearing_Hat").

  • Load 1,000 images from the CelebA test set via tensorflow_datasets (celeb_a, split 'test[:1000]').
  • Each image comes with 40 binary attribute labels. You will use Male as the prediction target and other attributes as subgroup slicers.
  • The classifier is MobileNetV2 fine-tuned on CelebA gender classification, loaded from a public checkpoint (details in Cell 2).

If tensorflow_datasets download fails in Colab, an alternative loading approach is described in Cell 2's notes.

Notes: Intersectional subgroups can be small. If any cell has fewer than 15 samples, note this in the output. Markdown cell: Did the intersectional analysis reveal a subgroup that performs worse than either individual attribute suggested? This is the core insight of intersectional auditing: combined identities can produce failures that single-attribute analysis misses.
