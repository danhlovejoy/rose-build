Module 3: Learned Embeddings

AIML 2013 • Introduction to Computer Vision • Spring 2026

HOG can tell you that two images have similar gradient patterns. MobileNetV2 can tell you that two images contain similar things. This week you strip the classification head off a network trained on 1.2 million images and use what's left to retrieve images by meaning, not by texture.

This Week's Focus

Extract 1,280-dimensional embeddings from MobileNetV2, build a similarity retrieval system, compare it directly against your HOG baseline on the same images, and visualize the embedding space with t-SNE.

What's in This Module

  1. Readings & Prep: 3Blue1Brown on neural networks, the Computerphile CNN walkthrough, Szeliski §5.4, and the Distill.pub Feature Visualization article. Together these explain what a neural network is, how convolutional layers build feature hierarchies, and what MobileNetV2 is actually doing when you call it in the lab.
  2. Module 3 Lab: Image Embeddings & Similarity Retrieval. Load MobileNetV2 with include_top=False, pooling='avg', extract embeddings for a set of images, build a cosine similarity retrieval system, compare against your Module 2 HOG baseline, and visualize the embedding space with t-SNE. No code scaffold this week: you write every cell using an LLM, document your prompts, and explain any fixes.
  3. Combined Lab: How Machines Represent Text and Images. For dual-enrolled students taking both CV and NLP. One notebook covering image embeddings and sentence embeddings side by side, submitted to both courses.

No code scaffold this week. The lab sheet gives you specification boxes: each one describes exactly what a cell must do, what it receives, and what it must produce. You give that specification to an LLM and write the code from its output. For each cell, include a markdown cell with the prompt you used. This is the workflow for the rest of the semester.
Tip: The readings build on each other. Watch the 3Blue1Brown video first — it introduces layers and weights, which the Computerphile video and Szeliski both assume you already understand. The Distill.pub Feature Visualization article is the most time-consuming, but it's also the most concrete: it shows you images that actually activate individual neurons, which is something you can point to in your demo.
