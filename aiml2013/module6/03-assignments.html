Deliverable 1: Module 6 Lab Lab

Overview

In the readings and in class, you learned how vision-language models generate captions by combining visual features with a language decoder, and how captioning hallucination is measured using metrics like CHAIR. This lab puts those ideas into practice. You will generate captions for 30 images using Gemini Vision, evaluate object accuracy against your reference descriptions using LLM-as-judge, compute per-category hallucination rates, and test how prompt wording affects what the model invents.

The hallucination gallery in Cell 6 is where the evaluation becomes tangible. You see the images, the invented objects, and the patterns. This is the artifact your presentation should center on.

What to Submit

  • A completed Colab notebook (.ipynb) saved to your GitHub repo at week7/lab6_captioning.ipynb.
  • All cells executed with visible outputs (don't clear outputs).
  • Each code cell preceded by a markdown cell with your LLM prompt and fix note.
  • No API keys or secrets anywhere in the notebook.
  • Submit the GitHub link in Canvas. If it's not in the repo, it doesn't exist for grading.
Start this in Thursday lab. We'll work through the captioning and evaluation pipeline together. Come with your images selected and reference descriptions written (Cells 1–2). We'll build the LLM-as-judge evaluation in Cell 4 together.

Deliverable 2: Module 6 Presentation Presentation

Overview

A three- to five-minute presentation during Tuesday's class (May 5, 7:00 PM). You may use your notebook, slides, or a combination. The goal is to communicate your findings, not to walk through code.

What to Cover

  1. Setup (30 seconds): What images did you use? How many categories? How did you write your reference descriptions?
  2. Hallucination findings: Show the per-category hallucination chart (Cell 5). Which categories had the most hallucinations? Show 2–3 examples from the hallucination gallery with the invented objects highlighted.
  3. Prompt sensitivity: Show the Cell 7 comparison. Did the cautious prompt help? By how much?
  4. Patterns: What types of objects does the model hallucinate? Are they random, or do they follow the scene-type patterns described in the CHAIR paper?
  5. Implications: If this model were generating alt text for blind users, would you trust it? What hallucination rate would be acceptable?

Format

Share your screen. You can present from your notebook, slides, or both. If you attend via Zoom, have your materials open and ready before your turn.
