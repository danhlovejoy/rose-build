Module 2: Hand-Crafted Features (TF-IDF)

AIML 2003 • Introduction to Natural Language Processing • Spring 2026

Before transformers, before word embeddings, NLP ran on features that researchers designed by hand. This week you'll build a TF-IDF sentiment classifier from scratch and pit it against Gemini to find out where simple math still holds up, and where it doesn't.

This Week's Focus

Build a TF-IDF + logistic regression sentiment classifier, compare it head-to-head against Gemini on the same test data, and visualize which words drive the model's predictions.
