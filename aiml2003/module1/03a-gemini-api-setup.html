Setting Up Your Gemini API Key

AIML 2003: Setup Guide. Free access to Google's LLM for the course

In this course we'll use Google's Gemini API to interact with large language models programmatically. The Gemini API has a free tier that requires no credit card and is more than sufficient for every lab in this course. This guide walks you through getting your API key and using it safely in Google Colab.

Part 1: Get Your Gemini API Key

Your API key is what lets your Python code talk to Google's Gemini models. Creating one takes about two minutes and requires no credit card.

1

Open Google AI Studio

Go to aistudio.google.com and sign in with your Google account.

2

Navigate to API Keys

In the left sidebar, click "Get API Key" (or look for the key icon).

3

Create your key

Click "Create API Key in new project". Google will generate a key that looks something like: AIzaSy...

4

Copy your key immediately

Copy the key and save it somewhere safe (a password manager or a private note, NOT a Google Doc or text file on your desktop). Google AI Studio may not show the full key again after you leave this page.

Treat your API key like a password: Anyone who has your key can make API calls that count against your quota. Do not share it, post it online, put it in a Google Doc, or paste it into a notebook cell. We'll store it securely using Colab's Secrets Manager in the next section.

Part 2: Store Your Key in Google Colab

1

Open Colab's Secrets Manager

In any Colab notebook, click the key icon in the left sidebar (or go to Tools → Secrets).

2

Add your Gemini API key

Click Add new secret and enter:

  • Name: GEMINI_API_KEY
  • Value: Paste your API key

Toggle "Notebook access" to ON for your current notebook.

3

Test it in a code cell

Run this code to verify everything is working:

!pip install google-genai -q

from google.colab import userdata
from google import genai

client = genai.Client(api_key=userdata.get('GEMINI_API_KEY'))

response = client.models.generate_content(
    model="gemini-2.0-flash",
    contents="Say hello to AIML 2003!"
)
print(response.text)

If you see a friendly greeting from Gemini, you're all set.

Here's what that did: You installed Google's google-genai Python library, loaded your API key securely from Colab's Secrets Manager so it never appears in your code, created a client connection, and sent a prompt to the Gemini 2.0 Flash model. You'll use this pattern in every lab.

Free Tier Limits

The Gemini API free tier is generous and more than enough for this course:

  • 15 requests per minute
  • 1,500 requests per day
  • No credit card required

For context, even the most intensive lab in this course will use fewer than 100 API calls. You won't come close to hitting these limits unless you're running code in an infinite loop.

The Model We'll Use

For most labs we'll use gemini-2.0-flash: it's fast, capable, and free. It handles classification, summarization, sentiment analysis, and all the tasks we'll tackle this semester.

Later in the course, you may experiment with other models (like gemini-2.0-flash-lite for speed or larger models for complex reasoning). We'll discuss trade-offs as they come up.

Common Questions

Do I need a credit card?

No. The free tier works without any billing setup. Do not enable billing unless you know what you're doing.

Can I use a different LLM (ChatGPT, Claude, etc.) for labs?

For graded labs, please use the Gemini API so we're all on the same platform and I can help debug issues. You're welcome to experiment with other models on your own time.

My key isn't working / I'm getting an error

Common fixes: (1) Make sure you copied the full key with no extra spaces. (2) Check that "Notebook access" is toggled ON in Colab's Secrets Manager. (3) Try creating a new key in Google AI Studio. If you're still stuck, bring it to Thursday lab.

I already have a Gemini API key from another project

You can reuse it. Just make sure it's stored in Colab's Secrets Manager with the name GEMINI_API_KEY.

What's the difference between the Gemini app and the Gemini API?

The Gemini app (gemini.google.com) is the chat interface (like ChatGPT's website). The Gemini API is the programmatic interface that lets you send prompts from Python code. In this course, we use the API so we can automate, iterate, and build real applications.
