Module 6: Simple Agents

AIML 2003 • Introduction to Natural Language Processing • Spring 2026

Until now, Gemini could only use what's in its parameters or what you fed it as context. This week you give it tools. The model decides when to call them, your code executes them, and the model incorporates the results into its response. This is how AI assistants work.

What's in This Module

  1. Readings & Prep: Gemini function calling documentation, an overview of the ReAct (Reason + Act) agent pattern, and a reading on the security risks of tool-using agents. Together, these explain how function calling works, why agents need a decision loop, and what can go wrong when you give an LLM access to external tools.
  2. Module 6 Lab: Simple Agents. Build a chatbot with three tools (calculator, date/time, knowledge base), implement the agent loop, test multi-turn conversations, analyze tool-routing accuracy, and probe error-handling edge cases.
  3. Assignment Sheet: Deliverable details, abbreviated rubrics with Canvas links, and the due date for Tuesday, May 5.
  4. Ethics Discussion: This week's ethics prompt on what happens when agents can act, not just answer.
