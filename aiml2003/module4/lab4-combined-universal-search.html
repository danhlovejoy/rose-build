The Task

In the readings and in class, you learned that neural embeddings encode meaning in a way that hand-crafted features cannot. This week you will build a single system that uses embeddings to index both text and images into one vector database.

You can query this database with text to find similar text passages, or query with an image to find visually similar images. The text and image embeddings live in different vector spaces and cannot be directly compared — this is not true cross-modal search. But you will see how a unified system can serve both modalities using independent embedding pipelines, and in the final cell you will build a crude bridge between the two using Gemini Vision.
