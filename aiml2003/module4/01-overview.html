What's in This Module

  1. Readings & Prep: A conceptual introduction to the attention mechanism (3Blue1Brown), the RAG architecture pattern, the ChromaDB quickstart, and a short reading on chunking strategies. Together, these explain why transformers work, how RAG uses retrieval to ground generation, and how to set up the vector database you'll use in the lab.
  2. Module 4 Lab: Chat with a PDF. Build a complete RAG pipeline: PDF text extraction, chunking, embedding, ChromaDB storage, retrieval, and Gemini-powered answer generation. Test it with factual questions, unanswerable questions (hallucination probes), and a chunk-size experiment.
  3. Combined Lab: The Universal Search Engine. For dual-enrolled students taking both NLP and CV. One notebook with a ChromaDB instance that indexes both text chunks and images, queryable in both modalities.
  4. Assignment Sheet: Deliverable details, abbreviated rubrics with Canvas links, and the due date for Tuesday, April 21.
  5. Ethics Discussion: This week's ethics prompt on who controls the context in a RAG system and what gets left out.
